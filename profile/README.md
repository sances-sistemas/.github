## 🚗 O que fazemos

Somos uma software house brasileira especializada no mercado automotivo. Nosso principal produto é o **SancesTurbo**, um ERP completo para concessionárias e revendas, cobrindo vendas, oficina, peças, financeiro e integrações com órgãos e sistemas do setor.

## 🛠️ Nossa stack

| Camada | Tecnologias |
|---|---|
| **Backend** | Node.js · AdonisJS · TypeScript |
| **Frontend** | React · Vite |
| **Banco de dados** | MySQL |
| **Infraestrutura** | Docker · GitLab CI/CD · AWS |

## 📦 Ecossistema

- **TurboWeb** — plataforma web moderna do SancesTurbo
- **Integrações** — RENAVE, Movidesk, webhooks e APIs do setor automotivo
- **Ferramentas internas** — automações, agentes e tooling de suporte
